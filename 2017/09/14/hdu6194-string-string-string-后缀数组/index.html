hdu6194-string string string 后缀数组

沈阳网络赛的题，其实很简单……然而比赛的时候把牛老板思路带歪了……今天下午就建模了……好多题还没补，补得题博客还没写……感觉又要gg了……

题目链接

题意:给出一个字符串，问有多少个子串恰好出现K次

思路：后缀数组，k个一组算lcp，这个是有可能重复的，减去上下K+1个的lcp就行了……讲道理比赛时候想的找height下降点也是对的，因为如果height不下降那么这k个的贡献就是零。
